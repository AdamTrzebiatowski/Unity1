Gra jest wynikiem pracy wyłącznie jednej osoby, nie jest pozbawiona błędów -
stanowi jedynie wersję poglądową. Zaimplementowane są podstawowe funkcjonalności,
jednak zdecydowanie nie jest to wersja ostateczna gry.

Krótka instrukcja:
WASD		- chodzenie
SPACJA		- skok
C		- kucanie
SHIFT		- sprint
L		- latarka

1		- wyjęcie broni
LPM		- strzał
PPM		- przybliżenie
R		- przeładowanie

ESC		- wyjście do pulpitu
